# README Assets

This folder stores visual media for the GitHub landing page.

## Suggested Files

- `hero-demo.gif`: Short 6-10 second loop showing end-to-end flow.
- `flow-animated.gif`: Optional animated architecture flow.
- `cover-static.png`: Static fallback image for low-bandwidth viewing.

## Notes

- Keep media truthful to actual behavior in this repository.
- Prefer compressed GIF/MP4 conversions to keep repo size practical.
- If you use external hosting, update links in `README.md` accordingly.
