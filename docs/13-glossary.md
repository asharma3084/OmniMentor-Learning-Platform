# 13. Glossary

- Evidence gating: Validation step that checks whether a claim is supported by available evidence artifacts.
- Rubric scoring: Structured scoring model that evaluates submission quality against explicit criteria.
- Blast radius: Estimated scope of impact caused by a change or failure.
- Dependency trace: Mapped relationship path between systems/components affected by a change.
- Flow A: Primary end-to-end learning workflow implemented in current scope.
- Smoke test: End-to-end runtime validation of critical path behavior.
- Ablation run: Comparative evaluation across retrieval modes.
- Synthetic-only: Data policy restricting repository datasets to non-personal, generated training content.
